# DroneHub – Startup Landing Website

DroneHub is a modern, responsive landing website designed to represent a fictional drone technology startup.  
The project demonstrates structured UI development, responsive design principles, and professional brand presentation.

---

## Overview

The website models how a real-world tech startup introduces its product and services online. It includes:

- Company introduction section
- Service offerings
- Portfolio showcase
- Call-to-action components
- Contact interface

---

## Key Features

- Fully responsive layout (desktop & mobile)
- Structured multi-section landing architecture
- Clean and modern UI design
- Optimized asset organization
- Lightweight and fast-loading implementation
- Easily extendable code structure

---

## Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- Bootstrap

The project can be deployed using GitHub Pages or any static hosting platform.
